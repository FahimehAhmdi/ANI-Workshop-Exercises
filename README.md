# ANI-Workshop-Exercises


# Advanced Neuroimaging & Neural Data Analysis (ANDA) Workshop - 2026

This repository serves as a comprehensive portfolio of my hands-on exercises, computational workflows, and data management pipelines completed during the 4-day **ANDA Workshop**. 

The core focus of this workshop is mastering the computational ecosystem required for handling large-scale neural data, implementing standardized data formats, and automating complex neuroimaging/electrophysiology workflows.

---

##  Repository Architecture & Daily Progression

The repository is structured chronologically, tracking advanced neural data engineering from raw data structures to automated multi-file workflows:

### 🔹 [Day 1: Computational Environments & Repositories](https://github.com/ANDA-NI-2026/NI-Day1-Code-and-Data-Repositories)
*   **Focus:** Setting up advanced computational environments, version control architectures for neural data, and handling initial workspace configurations.
*   **Key Tasks:** Foundational code organization and pipeline reproducibility.

### 🔹 [Day 2: Data Models & Electrophysiology with Neo](https://github.com/ANDA-NI-2026/NI-Day2-Data-Models)
*   **Focus:** Representing and manipulating complex neural datasets using the **Neo** framework.
*   **Key Tasks:** Working inside `03_exercises.ipynb` to handle electrophysiology data structures, trial segments, analog signals, and spike trains within a unified object-oriented Python data model.

### 🔹 [Day 3: Standardized Neural Data Formats](https://github.com/ANDA-NI-2026/NI-Day3-Data-Formats)
*   **Focus:** Exploring standard data formats critical for open-science reproducibility in neuroimaging and computational neuroscience.
*   **Key Tasks:** Metadata serialization, data conversion, and handling interoperable neuroscientific file specifications.

### 🔹 [Day 4: Multi-File Workflows & Pipeline Automation](https://github.com/ANDA-NI-2026/NI-Day4-MultiFile-Worfklows-with-Metadata-and-Snakemake)
*   **Focus:** Scale-up automation and professional workflow management.
*   **Key Tasks:** Working with `01_hw_exercises.ipynb` to organize metadata at scale and deploying **Snakemake** to build robust, reproducible, and automated multi-file computational pipelines.
